# AI MATH Project API

## Description
This is a web server project that serves a Web API and other methods to provide for the AI MATH application.

## Server Deploy: [Server](https://aimathprojectv2.azurewebsites.net/swagger/index.html)


## Instructions to run the project

### Before running this project, you must set up Visual Studio 2022 by following the link: [Visual Studio 2022](https://visualstudio.microsoft.com/vs/). Additionally, you need to install the ASP.NET and Web Development, as well as the .NET Desktop Development packages.

### After setting up all the necessary packages, you can run the project by following these steps:
1. Go to the `appsettings.json` file.
2. Contact me via Facebook or email to receive the password. Fill it in the "password" field under `ConnectionStrings`.
3. Press F5 to run the application.
